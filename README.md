# TypeSync
TypeSync - Typing Speed Tester

A modern and interactive typing speed tester built using React.js.
TypSync helps users test and improve their typing speed and accuracy through real-time feedback, timer-based challenges, and live character highlighting.

Features:
Real-time typing speed test
Live character highlighting
Typing accuracy calculation
WPM (Words Per Minute) tracking
60-second countdown timer
Random sentence generation
Result history tracking
Instant restart functionality
Responsive and clean UI

Tech Stack
React.js
JavaScript (ES6)
HTML5
CSS3
Vite

How It Works
A random sentence is displayed on the screen.
The timer starts automatically when the user begins typing.
Correct and incorrect characters are highlighted in real time.
Once the sentence is completed or time runs out:
Typing speed is calculated
Accuracy is calculated
Results are displayed
Previous results are stored in session history.
